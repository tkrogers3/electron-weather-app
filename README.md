<h1> Doppler 6000 </h1>
<h3> a simple electron.js weather application </h3>


<p> This application uses an API from <a href ="https://openweathermap.org/"> Open Weather Map</a> to display weather based by location. 
A user simply selects a zipcode, and the city, temperature in Fahrenheit, Celsius and Kelvin and weather condition are displayed. </p>
